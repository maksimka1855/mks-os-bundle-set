
# MKS‑OS

A minimal operating system prototype implemented as a single Python script.

## Overview

This is a lightweight educational OS prototype written in Python. It demonstrates basic OS concepts through a single executable script.

## Files

- `source/main.py` — Main OS source code (single script)
- `mks-os.exe` — Pre‑compiled executable (Windows)
- `README.md` — This documentation
- `LICENSE` — MIT license

## How to Use
run mks-os.exe

## Development
This project consists of a single Python script in source/mks-os.py

Edit source/mks-os.py
(Optional) Rebuild the executable using PyInstaller
pyinstaller --onefile source/mks-os.py --name mks-os
Note: PyInstaller must be installed (pip install pyinstaller).
